# node-sqlite-chart - HowTo
Example for
- reading data from file
- add data to sqlite-database
- show data in a chart

## cmd + url
node addData2sqlite.js

node importFile event.txt

node showChart.js 

test on: http://localhost:3005

## needed packages
  "chart.js": "^2.7.2",
 
  "express": "^4.16.3",
  
  "sqlite3": "^4.0.2"
